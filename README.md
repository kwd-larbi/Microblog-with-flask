# Microblog-with-flask

## Microblog with Flask

A lightweight micro‑blogging web application built with **Flask**, designed to help beginners understand routing, templates, forms, and basic web app structure. Users can create short entries and view recent posts through a simple, clean interface.

---

## 📌 Features

### **Add New Entry**
Submit short text posts through a form that sends data via **POST** to `/entry`.

### **Recent Posts Display**
Shows the latest entries with titles, timestamps, and content.

### **Semantic HTML Structure**
Uses clean, readable markup such as:
- `header`
- `main`
- `section`
- `article`
- `time`

### **Navigation Bar**
Includes placeholder links for future features like **Recent** and **Calendar**.

---

## 🧠 Purpose of the Project

This project is part of my learning journey in **Python**, **Flask**, and **web development**.  
It helps me practice:

- Building routes and handling form submissions  
- Structuring HTML pages  
- Connecting frontend and backend logic  
- Creating a real, functional mini‑blog
